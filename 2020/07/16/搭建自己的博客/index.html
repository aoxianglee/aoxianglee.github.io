前言

为什么要安装这个博客呢？有什么好处？

  1. 记录解决各种bug的方法，以免每次都去百度、Google。
  2. 记录和分享个人demo，方便后期的完善、项目的交流。整理思路，将知识融会贯通。
  3. 项目驱动力，保持博客的更新，会更有动力去做项目。
  4. 公司面试加分项
    ······

前期准备

  • github账号
  • 点击安装Nodejs
  • 点击git客户端

本机环境

  • Windows 10
  • node.js 12.18.2
  • hexo 3.1.0
  • git 2.27

创建hexo博客

1.建立blog文件夹（其他名字也行）

  • Win： Linux ： mkdir blog

2.生成博客

  • cd到blog文件夹
  • Win:  hexo init
  • Linux、Git Bush:  sudo hexo init

3.启动博客

Win、Linux：hexo sorhexo server

这时会生成一个局域网网址：

编写第一篇博客文章.md文件。

注：博客文件是用markdown格式编写的
